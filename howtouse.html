나만의 디스플레이형 즐겨찾기 만들기
이용법
 
  • [Type] 먼저 Type을 선택하세요.
    사이트 추가방식은 사이트를 추가하면 하단에 해당사이트가 오픈됩니다.
모두 추가하신 이후에는 내보내기 버튼을 통해 이름을 적고 저장하세요. 다음에 저장한 걸 해당 Type 형에서 "가져오기" 하면 저장한 내용 그대로 재현됩니다.
사이트 추가를 해도 하단에 페이지가 나타나지 않는 경우, 해당 사이트가 이를 거부(LOCK)하기 때문이니 이 사이트는 추가하지 마세요.
Live TV에서 국내 방송사들이 거의 이렇게 락을 걸어서 상단에 추가하지 못하고 있습니다.
특히 국내 유튜브 뉴스사이트들은
방송주소를 바꾸지 않는 미국(외국)과 달리 국내는 수시로 바꿉니다. 그래서 올리지 않고 있습니다.
PC나 태블릿에서는 잘 돼도, 핸드폰에서 안 되는 경우 역시 해당 사이트가 그 기능을 못하는 것이니 참고해주세요.
이럴 경우는 그림추가 방식이나 텍스트형(추가예정)을 이용해주세요.
  • 그림 추가 방식은 - 먼저 디렉토리를 클릭한 후,
    디렉토리 명을 적으세요.
    이후 사진추가를 누르면 디렉토리 선택, 사진주소, 이동할 페이지 주소를 넣을 입력창이 생깁니다.
    디렉토리를 선택 후 모두 입력한 후 추가 버튼을 누르면 해당 사진이 하단에 저장됩니다.
    그 사진을 클릭하면 해당 사이트로 이동하는 방식입니다.
    사진들은 드래그를 통해 순서도 바꿀 수 있습니다.
작업을 마치면 반드시 내보내기 버튼을 이용해서 저장하고, 이후 가져오기 버튼을 눌러 해당 파일을 선택해야 작업한 모습 그대로 재현되니 작업 후 내보내기 버튼 잊지 마세요!
TIP : 이미지 주소는
인터넷상의 이미지 주소 또는 직접 업로드한 이미지 URL을 사용할 수 있습니다.
링크 주소는 꼭 http(s)://로 시작하는 주소를 넣어주세요.
(폴더, 설명, 드래그 기능 없이
가장 간단하게 이미지를 관리할 수 있습니다.)
