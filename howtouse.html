나만의 디스플레이형 즐겨찾기 만들기
 이용법
 
  • [Type] 먼저 Type 을 선택하세요
    사이트 추가방식은  사이트를 추가하면 하단에 해당사이트가 오픈됩니다

         모두 추가하신이후에는 내보내기버튼을 통해 이름을 적고 저장하세요

          다음에 저장한걸 해당Type 형에서 - 가져오기를 하면

          저장한 내용 그대로  만들어집니다

 

           

  • 그림 추가방식은  -먼저 디렉토리를 클릭한후

         디렉토리 명을 적으세요

         이후 사진추가 를 누르시면

          디렉토리 선택  사진주소  이동할 페이지 주소를 넣을

          입력창이 생깁니다

          이때 디렉토리를 선택후 모두 입력한후  추가 버튼을 누르면

         해당 사진이 하단에 저장됩니다.

         그사진을 클릭하면 - 해당 사이트로 이동하는 방식입니다

         사진들은 드래그를 통해 순서도바꿀수 있습니다

 

          작업을 마치면 반듯이

 

          내보기버튼을 이용해서 저장을 하고

         이후 가져오기버튼을 눌러 해당파일을 선택해야

          작업한 모습 그대로 재현되니 - 작업후 내보기 버튼 잊지마세요

 

 

TIP : 이미지 주소는
인터넷상의 이미지 주소 또는 직접 업로드한 이미지 URL을 사용할 수 있습니다.
링크 주소는 꼭 http(s)://로 시작하는 주소를 넣어주세요.
(폴더, 설명, 드래그 기능 없이
가장 간단하게 이미지를 관리할 수 있습니다.)
