그림그리드 사용법
이미지 주소와 링크만 등록해서
간단하게 갤러리를 관리하세요!
  • [추가] 버튼을 눌러 이미지 주소와
    링크 주소를 입력한 뒤 저장하세요.
  • 등록된 이미지는 클릭하면 해당 링크로 이동합니다.
  • 각 이미지 우측 상단의 [수정] 버튼을 눌러
    이미지를 교체하거나 링크를 바꿀 수 있습니다.
  • 등록된 이미지는 자동으로
    그리드 형태로 보기 좋게 정렬됩니다,드래그를 해서 순서도 바꿀수 있습니다.
  • 브라우저를 끄기전 내보내기 버튼을 눌러야
    현재 만든 즐겨찾기 가져오기를 할수 있습니다.- 보통은 다운로드폴더에 있음
TIP : 이미지 주소는
인터넷상의 이미지 주소 또는 직접 업로드한 이미지 URL을 사용할 수 있습니다.
링크 주소는 꼭 http(s)://로 시작하는 주소를 넣어주세요.
(폴더, 설명, 드래그 기능 없이
가장 간단하게 이미지를 관리할 수 있습니다.)
