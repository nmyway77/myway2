즐겨찾기 추가

이용법

추가하기 — 맨 끝의 [+] 카드를 누르고 분류·이름·주소를 입력하면 등록됩니다.

분류 — 사이트 · 영상 · 사진 A~E 중에서 고릅니다. 보고 있던 탭이 자동으로 선택됩니다.

사이트 — 주소만 넣으면 제목과 도메인이 카드로 표시됩니다.

영상 — 유튜브 링크를 넣으면 썸네일이 자동으로 뜨고, 카드를 누르면 해당 영상으로 이동합니다.

사진 — 사진 주소(이미지 URL)를 넣으면 원본 비율 그대로 카드가 됩니다.

수정 / 삭제 — 카드 오른쪽 위의 색 모서리를 누르면 수정·삭제 창이 열립니다.

순서 변경 — 카드를 끌어다 놓으면 같은 분류 안에서 순서가 바뀝니다. (PC)

내보내기 / 가져오기 / 전체삭제 — 제목 옆 아이콘으로 백업·복원·전체삭제를 할 수 있습니다.

저장 — 모든 내용은 이 브라우저에 자동 저장됩니다.
