즐겨찾기 추가

사진 주소 (선택)
사진마다 사진 주소와, 그 사진을 눌렀을 때 이동할 페이지 주소(선택)를 적으세요. 맨 위 사진이 대표로 카드에 떠요.
내용 붙여넣기 (선택)
붙여넣은 내용은 이 항목의 전체 보기 화면에 사진들과 함께 나와요.

이용법

추가하기 — 맨 끝의 [+] 카드를 누르고 분류·이름·주소를 입력하면 등록됩니다.

분류 — 사진 A~E 중에서 고릅니다. 보고 있던 탭이 자동으로 선택됩니다.

사진 — 사진 주소(이미지 URL)를 넣으면 원본 비율 그대로 카드가 됩니다. 연결 주소를 넣으면 카드를 눌렀을 때 그 주소로 이동합니다.

수정 / 삭제 — 카드 오른쪽 위의 색 모서리를 누르면 수정·삭제 창이 열립니다.

순서 변경 — 카드를 끌어다 놓으면 같은 분류 안에서 순서가 바뀝니다. (PC)

내보내기 / 가져오기 / 전체삭제 — 제목 옆 아이콘으로 백업·복원·전체삭제를 할 수 있습니다.

저장 — 모든 내용은 이 브라우저에 자동 저장됩니다.

파일에 넣을 코드

아래 코드를 복사해 HTML 파일의 const DEFAULTS = { … }; 부분을 교체하면 영구 적용됩니다. 또는 편집용 HTML로 저장(편집 가능·코드 버튼 없음) / 보기전용 HTML로 저장(편집 불가)을 누르면 지금 내용이 담긴 완성 페이지를 받을 수 있어요.

가져오기 — 항목 연결

자동으로 알아보지 못했어요. 이 데이터의 어떤 항목을 주소·이름·사진으로 쓸지 골라 주세요.

네비(사진 탭) 편집

사진 탭의 이름을 바꾸거나 추가·삭제·순서를 변경할 수 있어요. (사이트·영상은 네비에 표시되지 않습니다.)
